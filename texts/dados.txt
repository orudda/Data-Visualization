Resultados de jogos da National Football League (NFL) desde 1966 com informações sobre probabilidades de apostas desde 1979. O conjunto de dados foi criado a partir de uma variedade de fontes, incluindo jogos e pontuações de vários sites públicos, como ESPN, NFL.com e Pro Football Reference. As informações meteorológicas são de dados NOAA com NFLweather.com uma boa referência cruzada. Os dados de apostas foram usados ​​em http://www.repole.com/sun4cast/data.html para as temporadas de 1978-2013. Os dados do Pro-football-reference.com foram então cruzados para linhas de apostas e probabilidades, bem como dados meteorológicos. A partir de 2013, os dados de apostas refletem as linhas disponíveis em sportsline.com e aussportsbetting.com.
